    - compareKorean("김치", "커피") returns false
    - pickDistractors(target, pool, 3): returns 3 items, none equal to target
    - pickDistractors prefers same conjugation_type as target
    - pickDistractors falls back to any vocab if same type pool too small
    - generateClozeQuestions(vocabPool, 10): returns up to 10 questions, each with sentence, blank position, correct answer, english hint
    - generateListeningQuestions(vocabPool, 10): returns up to 10 questions, each with target vocab and 4 shuffled options (1 correct + 3 distractors)
    - getUnlockedVocab(completedLessons): returns unique vocab items from all completed lessons
  </behavior>
  <action>
Create `src/utils/exerciseUtils.ts`:

1. `compareKorean(input: string, answer: string): boolean`
   - `input.trim().normalize('NFC') === answer.trim().normalize('NFC')`
   - Per RESEARCH.md Pitfall 1: Korean IME can produce different Unicode forms

2. `pickDistractors(target: VocabItem, pool: VocabItem[], count = 3): VocabItem[]`
   - Per RESEARCH.md distractor algorithm: prefer same conjugation_type, fallback to any
   - Filter out target, shuffle, slice to count
   - Use Fisher-Yates shuffle (inline helper, don't install a library)

3. `generateClozeQuestions(vocabPool: VocabItem[], count = 10): ClozeQuestion[]`
   - Type: `{ vocabId: string; sentence: string; blankSentence: string; correctAnswer: string; englishHint: string }`
   - For each question: pick a random vocab item, pick a random example sentence, replace the Korean word with "______" to create blankSentence
   - The correctAnswer is the vocab.korean
   - englishHint is the example.english
   - Shuffle pool, take up to count items (if pool < count, return what's available)

4. `generateListeningQuestions(vocabPool: VocabItem[], count = 10): ListeningQuestion[]`
   - Type: `{ vocabId: string; audioText: string; options: { text: string; isCorrect: boolean }[] }`
   - audioText = vocab.korean (the word to hear)
   - options = shuffle([target, ...3 distractors]) mapped to { text: v.korean, isCorrect }
   - Use pickDistractors for the 3 wrong options

5. `getUnlockedVocab(completedLessons: string[]): VocabItem[]`
   - Import LESSONS and VOCAB
   - Collect all vocab_ids from completed lessons, deduplicate, map to VocabItem objects
   - Return unique list

Create `src/utils/exerciseUtils.test.ts` with tests per behavior list. Use deterministic seeds or just test properties (e.g., "returns 3 distractors, none equal to target").
